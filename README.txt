Project Management course was an intensive 3-week course during the summer holiday. We learned about topics like Project purpose, how to communicate with stakeholders and people isnide the team, how to handle project complexity and the risks that uncertainty brings into the game.

We learnt how to effectively lead a project from the very beginning till the very end, including risk management and how to estiamte resources. We digged into different techniques how to schedule project's milestones and lead the project to be as effective as possible.

As our main output fro mthe course we analyzed Autonomous Shuttle Bus project driving at the DTU campus. Whether there were some mistakes which should have been done better can be found in the Final report.
